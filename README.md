This reflection tool contains four areas with reflection questions:
Strengths & skills, values & motivation, vision & goals, career fields, personality and development.
The tool can be used as an energiser or for reflection and personal development.
First, select the time using the timer. This allows you to control how much time is available for the answer.
Then select the desired area. It is possible to answer the questions in sequence or to have the questions displayed randomly using a dice.
